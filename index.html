SOBRE MI.

¡Hola! Soy Juan Diego, estudiante de Ingeniería en Sistemas con 7 semestres concluidos. Me considero una persona proactiva, comprometida y responsable, con 3 años de experiencia en el desarrollo de aplicaciones y páginas web diseñadas y programadas por mí; utilizando diversos lenguajes como JavaScript, Python y Java, además de manejar bases de datos tanto relacionales como no relacionales, programadas con MySQL y PHP. Soy un apasionado por el aprendizaje continuo y el trabajo en equipo. PROYECTOS.

JUST MUSIC

JAVA

Una aplicación de streaming de música cuyo concepto es ofrecer un espacio tanto para usuarios como para músicos, donde puedan compartir y descubrir nueva música de manera sencilla y rápida.

COMIC HOUSE

PHP HTML CSS

Una aplicación de streaming de cómics que permite a los usuarios leer sus cómics favoritos de manera sencilla y rápida. Hecho bajo HTML, PHP y CSS, Comic House permite buscar, leer y gestionar cómics de manera eficiente, utilizando una base de datos MySQL para gestionar los usuarios y los cómics disponibles.

FRISBY TRIVIA

JS HTML CSS

La Frisby Trivia es un proyecyto de juego personal, es na aplicación de trivia que permite a los usuarios poner a prueba sus conocimientos sobre música, cine y más temas juveniles y de cultura pop. Hecho bajo HTML, NODE.JS y CSS, Frisby Trivia ofrece una experiencia de juego interactiva y entretenida dirijida a jugarse en obs studio para imitar una transmision de quien quiere ser millonario.

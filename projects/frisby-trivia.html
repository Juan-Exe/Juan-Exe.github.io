Frisby Trivia

Ver en GitHub
Año 2024
Tecnologías JavaScript, Node.js, HTML, CSS
Descripción

Una aplicación de trivia interactiva que permite a los usuarios poner a prueba sus conocimientos sobre música, cine y más temas juveniles y de cultura pop. Hecho bajo HTML, Node.js y CSS, Frisby Trivia ofrece una experiencia de juego interactiva y entretenida dirigida a jugarse en OBS Studio para imitar una transmisión de quien quiere ser millonario.
